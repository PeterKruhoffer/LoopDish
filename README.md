## LoopDish

This is an [Expo](https://expo.dev) project created with [`create-expo-app`](https://www.npmjs.com/package/create-expo-app).

The majority of the code in this app was generated with gpt 5.2 codex

## Get started
This app requires you to have a convex account

1. Install dependencies

   ```bash
   bun install
   ```
2. Start convex dev

   ```bash
   bunx convex dev

3. Start the app

   ```bash
   bunx expo start
   ```
